# Free JSON Tools

A small website with simple JSON and CSV utilities.

## Pages

- /json-validator
- /json-formatter
- /json-minifier
- /json-to-csv
- /json-to-yaml
- /csv-to-json

## Features

- Validate JSON
- Format JSON
- Minify JSON
- Convert JSON to CSV
- Convert JSON to YAML
- Convert CSV to JSON
- Upload JSON files
- Download CSV files

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- json2csv
- js-yaml
