# Free JSON Tools

A small website with simple JSON utilities.

## Pages

- /json-validator
- /json-formatter
- /json-minifier
- /json-to-csv

## Features

- Validate JSON
- Format JSON
- Minify JSON
- Convert JSON to CSV
- Upload JSON files
- Download CSV files

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- json2csv
